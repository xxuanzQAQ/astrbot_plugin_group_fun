# astrbot_plugin_group_fun

AstrBot 群娱乐插件，提供天弃之子、同归于尽、精致睡眠三个随机禁言玩法。

## 环境要求

- AstrBot >= 4.13.0
- 平台：aiocqhttp（NapCat / Lagrange 等）
- Bot 账号须具有群管理员或群主权限

## 功能说明

### 天弃之子

发送 `/天弃之子`，Bot 从群成员中随机抽取一人禁言，禁言时长随机。

- 有一定概率闪电劈空，目标毫发无伤
- 有一定概率闪电余波波及召唤者，召唤者同样被随机时长禁言
- Bot 为管理员时，无法选中群主或其他管理员作为目标

### 同归于尽

发送 `/同归于尽 @群友`，将目标禁言，同时发起者自身受到倍数反弹禁言。

- 反弹倍数在 1 至 9 倍之间随机，低倍数概率更高
- 目标可通过 `禁止被同归` 拒绝被他人选中
- Bot 为管理员时，无法对群主或其他管理员执行禁言

### 精致睡眠

发送 `/睡眠套餐`，对自己执行一段时间的禁言（默认 8 小时）。

别名：`/精致睡眠`、`/来一份精致睡眠套餐`

## 命令列表

| 命令 | 权限 | 说明 |
|------|------|------|
| `/天弃之子` | 所有人 | 随机抽取一名群友禁言 |
| `/同归于尽 @群友` | 所有人 | 与目标一同被禁言，发起者受倍数反弹 |
| `/允许被同归` | 所有人 | 允许自己被他人同归于尽 |
| `/禁止被同归` | 所有人 | 拒绝被他人同归于尽 |
| `/睡眠套餐` | 所有人 | 对自己执行长时间禁言 |
| `/开启天弃之子` | 群管理员/群主 | 开启天弃之子功能 |
| `/关闭天弃之子` | 群管理员/群主 | 关闭天弃之子功能 |
| `/开启同归于尽` | 群管理员/群主 | 开启同归于尽功能 |
| `/关闭同归于尽` | 群管理员/群主 | 关闭同归于尽功能 |
| `/开启睡眠` | 群管理员/群主 | 开启精致睡眠功能 |
| `/关闭睡眠` | 群管理员/群主 | 关闭精致睡眠功能 |

## 配置项

在 AstrBot 管理面板的插件配置页中调整以下参数，修改后重启插件生效。

| 配置项 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| `tianqi_ban_min` | int | 60 | 天弃之子最短禁言时间（秒） |
| `tianqi_ban_max` | int | 600 | 天弃之子最长禁言时间（秒） |
| `tianqi_miss_prob` | float | 0.15 | 天弃之子劈空概率，范围 0~1 |
| `tianqi_self_ban_prob` | float | 0.2 | 天弃之子连带禁言召唤者的概率，范围 0~1 |
| `tonggui_ban_min` | int | 60 | 同归于尽最短禁言时间（秒） |
| `tonggui_ban_max` | int | 600 | 同归于尽最长禁言时间（秒） |
| `sleep_duration` | int | 28800 | 精致睡眠禁言时长（秒），默认 8 小时 |

## 权限说明

- 群主/管理员不会被随机选为天弃之子的目标（Bot 为管理员时）
- 群主/管理员无法被同归于尽目标（Bot 为管理员时）
- 群主/管理员无法使用精致睡眠（Bot 为管理员时）
- 各功能的开关命令仅群管理员和群主可操作，普通成员无效

## 数据存储

各群的功能开关状态和同归于尽白名单持久化存储于：

```
data/astrbot_plugin_group_fun/group_fun_state.json
```
